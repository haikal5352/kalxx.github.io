Tentang Saya

Halo, saya Haikal Aditama. Saya adalah mahasiswa Teknik Informatika di ULBI yang tertarik dengan pemograman, pengembangan aplikasi dan pembuatan game, saya senang mengeksplorasi teknologi baru untuk meningkatkan keterampilan saya di dunia IT.

Di proyek ini, saya membuat game "Pizzaman" yang seru banget.
